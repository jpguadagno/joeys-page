common core

core II studio • princeton university, 2021 • advised by prof. cameron wu & mónica ponce de león

Common Core is both a K-12 school and a community center, providing spaces for students as well as the broader community. By situating classrooms above shared program such as the library and gym, issues of access are centered on the vertical circulation cores. These cores each gain a mezzanine which conforms to the program below and acts as a threshold for the school above.
